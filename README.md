# AETRUX

Signal-first digital portfolio built for motion, contrast, and narrative web experiences.

![Build](https://img.shields.io/badge/build-passing-0f172a?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-16a34a?style=flat-square)
![Version](https://img.shields.io/badge/version-1.0.0-2563eb?style=flat-square)

## Features

- Motion-heavy, cinematic section transitions
- Responsive React + TypeScript architecture
- Componentized UI with Radix and shadcn patterns
- Media-driven storytelling with video assets

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

Open: `http://127.0.0.1:4444`
